之前我写过一个插件，是用来将 MasterGo 文件转换到 Figma 的。

是很久之前开始做的，那个时候甚至还没有 ChatGPT 3.5，纯手搓的代码。

一开始的想法很纯粹，就是试图对 MG 导出的 .mg 文件进行逆向的解析，解析出 Figma 的文档格式，然后通过 figma 的插件逐渐地导入到 Figma 文件中。但是很快就放弃了，.mg 文件中对于图层的存放使用的是专有二进制，对当时的我来说几乎不可能破解（现在好像不靠 AI 也不行）。

然后我退而求其次，在 MasterGo 中也做了一个插件`SendToFigma`，通过读取界面上的图层，转成 json 文件，然后再导入到 figma 中的另一个插件`ReceiveFromMasterGo`进行导入并还原，由此实现文件转换。这是我后面很长一段时间的技术方向。

不过也很容易想到，这个方向太吃人力了，MG 的图层格式只是看上去和 Figma 一样，实际上每种图层总有那么一二三八九个差异点，光靠人力真的太吃力了。所以之前我是做了几个月就放弃了，虽然也靠人力完成了近半的图层类型解析。

直到今年，codex 前几个月开始额度重置大放送，让我刚好有很多多余的 token，就开始继续写这个插件了。

我让 AI 先从我之前写到一半的插件里，总结出 MG 导入到 Figma 可能的坑，先做一个初版的 Workflow。还好之前基础打得好，AI 很快就搞定了这个事。

之后就是准备测试集了，我让 Codex 去连接到 Figma，生成它认为尽可能全面的测试用图层，涵盖所有可能的 figma 图层类型。之后用 MG 的官方导入 Figma 功能把这些图层导入，再用`SendToFigma`导出 JSON 到 `ReceiveFromMasterGo`将图层再回到 Figma。

稍微有点晕，一个简单的版本可以理解成，Codex 自己生成测试集用来测试 JSON 还原方式的效果。

我让 Codex 自己对比测试集前后的变化，迭代了十几轮吧，算是把几乎所有图层都搞定了。这事本来也应该差不多到这就 OK 了。

但是后来又发生了一个问题，我把插件给同事用，他的 MG 文件非常大，导致`SendToFigma` 这个插件崩溃了！Codex 查不出异常，后来手动 review 后发现。MasterGo 给插件运行分配了固定大小的沙箱，而且这个沙箱的内存不会也不能释放。导致不断导出图层的过程中内存溢出，触发 OOM 问题。

这个事情几乎就无解了，也是在这个时候我开始尝试另一个方向：Figma 插件 `ReceiveFromMasterGo`直接导入 .mg 文件。这样绕过 MG 的插件环境，按理来说能解决。

所以我之前的设想是，让 GPT 开 Extra High Thinking，去找出 .mg 中二进制和真实数据之间的关系，从而将无法人力解读的 01 直接转成 Figma 图层。

对，失败了，二进制数据非常碎而且长，GPT 读取后设计的转换规则极易遗漏图层的细节，通过二进制直接还原的图层和原图层差异实在过大，几乎不可用。

Opus 也一样，即便我还手动开了 1M 上下文，也只是比 GPT 5.5 “稍好一些”。所以本来我没有对 Fable 有什么指望的。

直到昨天，我拿之前 GPT 的测试集，让 Fable 做了同样的事，结果近乎完美。

我给 Fable 了这些上下文：之前的方式导出的 JSON 文件，通过 JSON 导入到 Figma 的文件的链接，还有 .mg 文件。跑了半小时吧，当然在之前我为了节省 token 拿 cowork 细化了一下 prompt，都加起来应该不到一小时。全程我没有干预，Fable 自己在解读二进制，和 JSON 和 Figma 图层交叉对比，跑 subagents 查看可能遗漏的地方以及定位对应二进制串。

最后第一轮的成果是，我手动验出了四个遗留的问题（非常小，乍一看都看不出来那种），其余的还原效果和 JSON 还原一样。

之后，我再指出四个遗留的问题，并要求 Fable 找出相似的情况。我看思考过程，Fable 已经在开始将这四个问题和上轮中遗漏未处理的二进制片段做对比了，并不断迭代直到几乎没有遗漏的二进制碎片。换句话说，把 .mg 文件榨干了才结束。

二轮的处理非常完美了，除了刚才指出的四个问题，通过上述对二进制的“榨取”，Fable 还额外破解了十几个类似的二进制编码规则。这样转换后的图层就完全和 MG 原文档中的一样了。

Anyway，到现在我的这个插件应该才算是将近结束了，之前的 JSON 导出格式应该已经完成了它的使命（除非还有测试集覆盖不到的内容，或者 MasterGO 把二进制解码规则再改了），剩下应该只剩下调一下 UIUX 让其更加易用一些了。

有需要的 GitHub 上直接搜索 **MasterGo2Figma** 吧，已经 Push 了。
